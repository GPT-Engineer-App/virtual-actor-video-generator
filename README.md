# virtual-actor-video-generator

A website where you can input a text prompt and give some details about which virtual actor you'd like to read said text and it would give you a video of this actor saying what the text says. The website would automatically generate this video by connecting to different AI tools.

## Setup

```sh
git clone https://github.com/GPT-Engineer-App/virtual-actor-video-generator.git
cd virtual-actor-video-generator
npm i
```

```sh
npm run dev
```

This will run a dev server with auto reloading and an instant preview.
